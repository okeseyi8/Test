Q3 - Break and Continue

Break - is used when we want to stop a loop from running, stating that the loop should stop where we added the break .
Continue - is used when we want to skip some parts of the code but continue with the rest of the code.

const numbers = [1,2,3,4,5];

for(let num of numbers) {
  if(num === 3) break;
  console.log(num);
}

for(let num of numbers) {
  if(num % 2 === 0) continue;
  console.log(num);
}

Explain questions - 
Q1 - Hoisting is a behaviour in javascript whereby a function is called even before declaring the function and the code runs.
Hoisting in function declaration works because the function is hoisted directly and we can even call the function before it appears in our code.
Hoisting in function expression only works when the function is called after the initialization of the function but won't work before initialization because in function expression the function is stored in a variable and only the variable is hoisted then so calling the function before initialization won't work and the code will crash even before going to the next line.

Example
Hoisting in Function Declaration
  announcement();
  function announcement() {
    console.log("Welcome to class!");
  }

Hoisting in Function expression
  const announcement = function() {
    console.log("Welcome to class!");
  }
  announcement();
